# Windowee Backend

Δημιουργήθηκε με bun και είναι πλήρως συμβατό με NodeJS.

Πριν το τρέξεις πρέπει:
- να συμπληρώσεις το αρχείο στο `config/firebase.json` ώστε να στέλνει push notifications.
- να αλλάξεις τα στεράκια `***` στον .env (αντέγραψε το .env.example στο .env).
- να στήσεις την βάση που βρίσκεται στο φάκελο database.

Ιδανικά κάνε το build με bun `bun run build` και τρέξε το `dist/index.js` σε όποιον NodeJS - compatible server θέλεις.
